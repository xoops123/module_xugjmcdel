xugjメニューリフレッシュ（xugjmcdel）
=====================================
バージョン 0.05
ライセンス　GPL
作者　domifara
掲載者　marine
=====================================

domifaraさん作のモジュールで、その名のとおりxugj_assignにより作成されたメニュー項目用のキャッシュファイルを削除するものです。

私、marineが作成したテーマでは、xugj_assignを利用してメニューを自動でメニューを表示するようになっていますが、メニュー項目を編集したり、モジュールを追加・削除した場合、ＦＴＰでキャッシュファイルを削除しないと、メニュー表示が更新されません。

そこで、このモジュールを使うと管理画面でキャッシュファイルを削除することができるので、便利です。

2012.7.17現在、domifaraさんのサイトが閉鎖中で、ダウンロードページが参照できないため、ここに掲載させていただくことにしました。
